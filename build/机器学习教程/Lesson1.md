# 数学分析
## 导数与梯度

机器学习的特征，随着任务的不断执行，经验的累积会带来计算性能的提升
主要分类为：有监督学习，无监督学习，增强学习
机器学习中可以解决的问题：
1、 给定数据的预测问题
    数据清洗、特征选择
    确定算法模型、参数优化
    结果预测
2、 不能解决的问题
    大数据存储、并行计算

机器学习的一般流程：
数据收集、数据清洗、特征工程、数据建模
